# Peacock Promotors - Luxury Real Estate

A premium real estate application built with React, GSAP, and Tailwind CSS. This project showcases advanced styling techniques, including glassmorphism, complex gradients, and smooth scroll animations.

## Features

- **Advanced Animations**: Powered by GSAP (GreenSock Animation Platform) for timeline controls, scroll triggers, and smooth transitions.
- **Modern Styling**: Utilizes Tailwind CSS for utility-first styling and `shadcn/ui` concepts for component architecture.
- **Visual Effects**: Includes backdrop blur (glassmorphism), animated gradients, and lucide-react icons.
- **Responsive Design**: Fully responsive layout optimized for all device sizes.

## Tech Stack

- **React**: Frontend library.
- **GSAP**: Animation engine.
- **Tailwind CSS**: Styling framework.
- **Lucide React**: Icon set.
- **Sonner**: Toast notifications.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

2.  **Start the Development Server**
    ```bash
    npm start
    # or
    yarn start
    ```

3.  **Build for Production**
    ```bash
    npm run build
    # or
    yarn build
    ```

## Project Structure

- `src/components/`: Reusable UI components (Hero, About, Projects, Contact).
- `src/components/ui/`: Base UI components.
- `src/mock.js`: Mock data for properties and company info.
